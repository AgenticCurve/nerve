⏺ Yes! There's nerve extract for testing the parser. Here's how to use it:

  # Extract from a file
  nerve extract /path/to/output.txt

  # Extract from stdin
  cat /path/to/output.txt | nerve extract

  # Output as JSON (useful for seeing structure)
  nerve extract --json /path/to/output.txt

  # Show only raw response
  nerve extract --raw /path/to/output.txt

  # Show only last section
  nerve extract --last /path/to/output.txt

  # Full content (no truncation)
  nerve extract --full /path/to/output.txt

  # Parse Gemini output instead of Claude
  nerve extract --type gemini /path/to/output.txt

  The command uses the Claude parser by default (ParserType.CLAUDE). It parses the content and shows structured sections (text, tool calls, etc.).
